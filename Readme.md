## Code  
### 1xx  Controller
>100 连接登录请求    

### 2xx  Client
>200 连接登录请求  
>210 用户注册请求  

### 3xx  Server
>300 允许登录  
>301 拒绝登录  
>310 用户注册成功  
>311 用户注册失败  

### 4xx  Error
>400 未定义错误  
>410 缺少code  
>411 缺少必要字段  
>420 数据库错误  


### 500 Other
>500 数据长度通知
